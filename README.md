# 🌍 AI Language Translator

A modern AI-powered Language Translator built using **HTML, CSS, and JavaScript**. This web application allows users to translate text between multiple languages with a clean and responsive user interface.

## 🚀 Features

- 🌐 Translate text between multiple languages
- 🔄 Swap source and target languages
- 📋 Copy translated text
- 🔊 Text-to-Speech
- 🗑 Clear input and output
- ⌨ Character Counter
- ⚡ Keyboard Shortcut (Ctrl + Enter)
- 📱 Responsive Design
- 🎨 Modern Glassmorphism UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- MyMemory Translation API

## 📂 Project Structure

```
CodeAlpha_LanguageTranslator
│── index.html
│── style.css
│── script.js
│── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Run `index.html` using Live Server.
4. Enter text and click **Translate**.
